MDK5 常用快捷键

## 1. 调试类

| 序号 | 快捷键                | 功能                   |
| ---- | --------------------- | ---------------------- |
| 1    | **F7**                | 编译（Build)           |
| 2    | **F8**                | 下载（Download）       |
| 3    | **Ctrl + F5**         | 开始调试/退出调试      |
| 4    | **F10**               | 执行 过去（Step Over） |
| 5    | **F11**               | 执行 进去 （Step）     |
| 6    | **Ctrl + F11**        | 执行 出去 （Step Out） |
| 7    | **F5**                | 执行到断点处           |
| 8    | **F9**                | 插入断点 / 取消断点    |
| 9    | **Ctrl + Shift + F9** | 取消所有断点           |
| 10   | **Ctrl + F10**        | 执行到光标处           |

## 2. 跳转类

| 序号 | 快捷键               | 功能         |
| ---- | -------------------- | ------------ |
| 1    | **Ctrl + Shift + F** | 全局查找     |
| 2    | **Ctrl + G**         | 跳转到指定行 |
| 3    | **Ctrl + H**         | 替换字符串   |
| 4    | **Ctrl + F**         | 当前文件查找 |

## 3. 编辑类

| 序号 | 快捷键                               | 功能                                 |
| ---- | ------------------------------------ | ------------------------------------ |
| 1    | **Ctrl + L**                         | 删除当前行                           |
| 2    | **Ctrl + 左右方向键**                | 快速移动光标                         |
| 3    | **Ctrl + 上下方向键**                | 上下移动屏幕                         |
| 4    | **Ctrl + Shift + 左右方向键**        | 左右移动并选中文字                   |
| 5    | **Ctrl + E**                         | 在配对的括号跳转                     |
| 6    | **Ctrl + U**                         | 将选中的区域中的英文字母全部换成小写 |
| 7    | **Ctrl + Shift+ U**                  | 将选中的区域中的英文字母全部换成大写 |
| 8    | **alt + Shift + 方向键或者鼠标左键** | 以光标所在地进行自由选择             |
| 9    | **Ctrl + I**                         | 匹配式查找                           |
| 10   | **Ctrl + F6**                        | 切换标签                             |
